# AGENTS.md

## Project
ATP Knowledge Studio is a desktop AI research and writing studio for marketing academic articles, textbook chapters, Obsidian knowledge notes, slide briefs, and visual infographic outputs.

## Product Direction
The app uses an 8-bit pixel virtual office interface inspired by retro school-office aesthetics. Do not copy copyrighted characters, official game assets, or recognizable game scenes.

## Tech Stack
- Tauri v2
- React
- TypeScript
- Tailwind CSS
- Mock local data in Sprint 0
- Future integrations: OpenAI, Gemini, Obsidian, NotebookLM, Canva

## Repository Rules
- Work from the repository root.
- Do not create a nested duplicate project folder.
- Keep a scalable structure: components, pages, layouts, features, data, types, styles.
- Do not hardcode API keys.
- Use .env.example for future environment variables.
- Use mock data for Sprint 0.
- Report all build or install errors clearly.

## UI Rules
- Desktop-first layout.
- Dark blue dashboard base.
- Accent colors: light blue, teal green, golden orange.
- Pixel-inspired UI, but text must remain readable.
- Virtual office must be functional, not decorative.

## Writing/Product Rules
- Default writing output is formal Thai with English technical terms in parentheses.
- APA 7 citation orientation.
- No fabricated citations or fabricated case studies.
- Case studies must be verifiable.
