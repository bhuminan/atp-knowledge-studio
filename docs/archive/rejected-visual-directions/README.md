# Rejected Visual Directions Archive

This folder contains historical ATP visual direction documents that are no longer active.

These files are retained for audit and history only. They must not be used as implementation guidance for new UI work.

The accepted UI baseline is the Win95 Functional Frontstage:

```text
docs/product/ATP_WIN95_FRONTSTAGE_STYLE_BASELINE.md
```

Rejected visual directions include 8-bit pixel art virtual office, isometric virtual studio, dashboard lobby scene, PNG banner dashboard, SVG/isometric scene dashboard, agent-room visual office, and visual-dashboard-heavy frontstage papers.

Future UI work should preserve the current Win95 Functional Frontstage visual system unless a later approved product brief explicitly changes it.
