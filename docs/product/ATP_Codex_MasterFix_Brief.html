Section 01 — Most Critical

Windows 95 Authentic Color System

Fix every color violation first — before touching anything else

Current build uses wrong colors throughout. These values come directly from Windows Registry HKEY_CURRENT_USER\Control Panel\Colors. Apply globally to every component.

The One Rule — Memorise This

Text color has only 3 values in the entire app

#000000 black — on every light background (silver, white, yellow, cream). This is the default for ALL text.
#ffffff white — ONLY on navy #000080 background (titlebar and selected/highlighted items).
#808080 gray — ONLY for disabled/inactive elements. Never for normal body text, labels, or descriptions.
